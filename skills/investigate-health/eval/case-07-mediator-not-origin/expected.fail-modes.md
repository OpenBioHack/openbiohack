# Failure modes this case tests

**Primary:** picking a graph-upstream node (low ferritin, low fT3,
post-viral history) as "integrator" because it's upstream — without
checking weeks-scale intervention-leverage on quality-of-life-affecting
symptoms.

**Secondary:** defaulting to a familiar mediator class (immune /
mast-cell) when the actual mediator-node passing all four gates is a
different physiology entirely (autonomic).

Enforces:
1. Integrator identified by the four-gate mediator-node test
   (convergence + emergence + intervention-leverage + cheap-testability),
   not by graph-position or by mediator-class familiarity.
2. Autonomic dysregulation (or equivalent autonomic-mediator
   description) named as the integrator candidate.
3. A cheap reversible discriminator appears (paced breathing + salt +
   compression, vagal-pacing trial, or equivalent) — readable on
   weeks-scale symptom diary.
4. Upstream findings (ferritin, fT3) are addressed separately as
   origins/substrate, NOT named as the integrator.

— the mediator-node definition" (Round 5).
