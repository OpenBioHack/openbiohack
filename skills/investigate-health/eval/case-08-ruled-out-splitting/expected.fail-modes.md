# Failure modes this case tests

**Primary:** collapsing a diagnostic-label rule-out by criteria-failed
into a process rule-out, then dropping the underlying mechanism.

Enforces:
1. Memory rule-out classified by category (a/b/c) before being treated
   as binding.
2. For (b) or (c), Q1-Q5 atypical-presentation gate is walked, with the
   result recorded in `candidate-ledger.md` as a `ruled-out-gate-result`
   field.
3. The underlying process (peripheral / tissue-level thyroid hormone
   availability deficit) re-entered as a T3 mediator candidate.
4. The label stays ruled out by criteria; the mechanism is back in
   play.
5. Cheap reversible discriminator (low-dose T3 trial, or focused
   rT3/fT3 panel, or upstream-driver removal) is named.

Linked SKILL.md rule: register §"Rule-outs are typed, not blanket" +
Step 3 §"Atypical-presentation gate (Q1-Q5)" (Round 5). Case uses a
non-immune domain to test that the rule generalises beyond mediator
classes the procedure has seen before.
