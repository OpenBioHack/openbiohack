# Corrections Log

**Purpose**: Document every mistake, wrong assumption, or incorrect
conclusion discovered during this process. This file prevents us from
making the same mistake twice.

This is not a shame file — it's a learning file. Every correction makes
the model more accurate.

---

## How to Use
When you discover that something you believed or recommended was wrong:
1. Add an entry below with the date
2. Document what was wrong, why, and what's correct
3. Add a brief note on what mental model needs to change
4. Check if any current recommendations were based on the wrong information
   and update them
